# Электронный сервисный паспорт техники Silant

## Описание проекта

Данный проект представляет собой веб-приложение для ведения электронного паспорта складской техники Silant.

Система позволяет хранить информацию о машинах, техническом обслуживании и рекламациях, а также предоставляет доступ пользователям с разными ролями.

## Реализованный функционал

В проекте реализованы:

- регистрация и авторизация пользователей;
- разграничение доступа по ролям:
  - Manager;
  - Client;
  - Service;
- просмотр списка техники;
- просмотр информации о конкретной машине;
- просмотр технического обслуживания;
- просмотр рекламаций;
- фильтрация данных;
- REST API для получения данных.

## Структура проекта

Основные приложения:

- `machines` — информация о технике;
- `maintenance` — техническое обслуживание;
- `complaints` — рекламации;
- `references` — справочники;
- `accounts` — работа с пользователями;
- `api` — API проекта.

## Используемые технологии

- Python 3
- Django
- Django REST Framework
- django-filter
- SQLite

## Установка и запуск проекта

Клонировать репозиторий:

```bash
git clone https://github.com/koracarkade/silant.git
```

Перейти в папку проекта:

```bash
cd silant
```

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать виртуальное окружение:

Windows:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Выполнить миграции базы данных:

```bash
python manage.py migrate
```

Загрузить данные справочников:

```bash
python manage.py loaddata fixtures/references.json
```

Запустить сервер:

```bash
python manage.py runserver
```

После запуска проект доступен по адресу:

http://127.0.0.1:8000/
